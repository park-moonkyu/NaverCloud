## 개요
네이버클라우드플랫폼의 다양한 서비스를 공유합니다.<br>
다양한 아키텍처를 공부하면서 실제 많이 활용되고 있는 서비스간의 연계를 통해 해당 레포지토리를 보시는 분들에게 정보를 제공합니다.<br>
